## cURL example

Lists information on cats.

### cURL command

```shell
curl http://localhost:3000/cats/1 
```

### cURL response

```shell
 
{
    "Breed": "Main Coon",
    "Life Span": "9-15 years",
    "Coat": "Dense double coat",
    "Color": "black",
    "id": 1
  }
```
