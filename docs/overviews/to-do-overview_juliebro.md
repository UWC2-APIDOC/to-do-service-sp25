## ✓ Effortless to-do list integration

Integrate task management into your apps with the to-do service. You can [**create**](https://uwc2-apidoc.github.io/to-do-service-sp25/tutorials/add-a-new-task/), [**prioritize**](https://some-link-here/), and [**remind**](https://uwc2-apidoc.github.io/to-do-service-sp25/api/task/) with just a few RESTful API calls. The service and your data is [**securely hosted**](https://some-link-here/) in the to-do service private cloud.

*Example of JSON used to create a new user with the POST method and an endpoint of `{base_url}/tasks`*

```json
{
    "user_id": 23,
    "title": "Set up Zoom meeting with web team",
    "description": "Kick off new product release campaign",
    "due_date": "2025-05-19T14:00",
    "warning": "-10"
}
```
